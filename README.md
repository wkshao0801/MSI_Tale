# MSI_Tale
Mass Spectrometry Imaging tool for Tissue Alignment, Labeling, and Export  
  
[![License](https://img.shields.io/badge/license-BSD_3_Clause-orange)](https://github.com/wkshao0801/MSI_Tale/blob/main/LICENSE)
[![Latest Version](https://img.shields.io/badge/release-v2.7.2-blue)](https://github.com/wkshao0801/MSI_Tale/releases/tag/v2.7.2)

## System Recommendations
MATLAB 2023a (recommended; compatibility with earlier versions is not guaranteed)

## Installation
1. Download the latest release from the [Releases page](https://github.com/wkshao0801/MSI_Tale/releases/tag/v2.7.2).
2. Install the [MATLAB Runtime R2023a (9.14)](https://www.mathworks.com/products/compiler/matlab-runtime.html). 
3. Install the [MSI_Tale_2_7_2.exe](https://github.com/wkshao0801/MSI_Tale/releases/download/v2.7.2/MSI_Tale_2_7_2.exe).

## Usage
1. For detailed instructions on using MSI Tale, please refer to the [User Guidelines](./User_guidelines%20(v2.7.X).pdf). 
2. For a step-by-step tutorial, please refer to the instructional video: [YouTube](https://youtu.be/08Tsj9h0GM4). 
3. Testing data can be found in [Example](./Example).
